### 프로젝트 개요
이 프로젝트는 **이커머스 플랫폼(예: 쿠팡, 아마존)** 의 리뷰 데이터를 수집해,  
허위 후기(Fake Review)를 탐지하는 모델을 구현한 것입니다.  
텍스트 전처리와 TF-IDF 기반 특징 추출, 여러 분류 모델(Logistic Regression, SVM 등)을 비교했습니다.

### 사용 기술
- Python  
- scikit-learn  
- Selenium (데이터 수집)  
- Pandas
